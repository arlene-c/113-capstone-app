# Asset Sources

## ASL Fingerspelling Letter Images

- Local path: `public/asl-letters/*.svg`
- Source collection: Wikimedia Commons `Sign language <letter>.svg`
- Original source note on Commons: public domain images from `wpclipart.com`

Reference pages:
- https://commons.wikimedia.org/wiki/Category:ASL_letters
- https://commons.wikimedia.org/wiki/File:Sign_language_A.svg
